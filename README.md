### CINNAMON GAMES

# ![pageres](src/images/logo.png =80x)

Website to show released games and games in development


## Install

Clone the repo then run:

```
$ npm install 
```

## Running local instance

```
$ npm run start
```

This will start the webpack server so you can see the webpage in
`localhost:8000`
